# College FeePayr 🎓💳

A PHP-based web application for managing college fee payments, student records, and payment receipts.

![Project Preview](https://via.placeholder.com/800x400?text=FeePayr+Screenshot) <!-- Replace with actual screenshot -->

## Features ✨
- **Student Dashboard**: View fee details and payment history.
- **Clerk Panel**: Manage students, courses, batches, fee structures and can also make fees payments.
- **Clerk Panel**: Manage students, clerks and can view the fees details.
- **Payment Integration**: Razorpay
- **Receipt Generation**: Auto-generate PDF receipts.

## Technologies Used 🛠️
- PHP 8+
- MySQL Database
- HTML5/CSS3
- JavaScript (jQuery)
- Razorpay API
